# Minecraft-project
Minecraft data used for a final project in my Data Visualization class.

Excel file of Minecraft data. 

Player_by_system is data of active player counts based on which system the player is using taken from the blockbase-tools website. 

Bugs is an acculumation of the number of bug fixed during every single Minecraft release update. This data was taken from every version update page on the Minecraft Wiki. 

Speed is taken from the speedrun website. To ensure that the data would be as equal as possible, the Minecraft Java Edition results were filtered using the following terms: any % glitchless, random seed, 1.16-1.19, vanilla, and difficulty set to easy. For Minecraft Bedrock Edition, results were filtered using the following terms: any % glitchless, random seed, 1.16.100-1.171.18+, PC, and difficulty set to easy. The top 50 player times for both editions were combined to create the dataset. 

Session_stats is data of the player average daily session time based on their Minecraft game experience. This data was taken from the blockbase-tools website. 

Server is data taken from the Minecraft official server list. The search was filtered by edition and sorted by player count high -> low. If a server didn't have a version number associated with the page, it was not included in the dataset.
